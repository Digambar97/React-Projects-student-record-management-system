# React-Projects-student-record-management-system
A Student Management CRUD application with REST API integration, built using React, Express, and MySQL. Features include adding, updating, deleting, and displaying student records in real-time



🎓 Student Record Management System

A full-stack CRUD (Create, Read, Update, Delete) application to manage student records.
Built using React.js, Node.js, Express.js, and MySQL.

🚀 Features
➕ Add new student records
📋 View all students
✏️ Update existing student details
❌ Delete student records
🔄 Real-time UI updates
🌐 RESTful API integration
🛠️ Tech Stack
Frontend
React.js
Axios
CSS
Backend
Node.js
Express.js
Database
MySQL
